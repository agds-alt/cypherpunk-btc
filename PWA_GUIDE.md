## 📲 How to Install

### **Android (Chrome/Edge)**

1. Open https://your-domain.com in Chrome
2. Tap **3 dots menu** (⋮) → **Add to Home screen**
3. Confirm by tapping **Add** or **Install**
4. App appears on your home screen with Bitcoin icon (₿)

**Alternative:**
- Look for "Install App" banner at bottom of page
- Tap **Install** when prompted

### **iOS (Safari)**

1. Open https://your-domain.com in Safari
2. Tap **Share button** (□↑) at bottom
3. Scroll down → tap **Add to Home Screen**
4. Name it (default: "Cypherpunk BTC") → tap **Add**
5. App appears on your home screen

**Note:** iOS requires Safari browser for PWA installation.

### **Desktop (Chrome/Edge)**

1. Open https://your-domain.com
2. Look for **install icon** (⊕) in address bar
3. Click → **Install**
4. App opens in its own window

## 🧪 Testing PWA Locally

### **1. Build Production Version**
```bash
pnpm build
pnpm start
```

### **2. Access on Mobile**
```bash
# Find your local IP
ip addr show | grep "inet " | grep -v 127.0.0.1

# Or on macOS
ifconfig | grep "inet " | grep -v 127.0.0.1
```

Then on your phone (same WiFi):
```
http://YOUR_IP:3000
```

Example: `http://192.168.1.100:3000`

### **3. Test PWA Features**

**Installation:**
- Chrome DevTools → Application → Manifest (check for errors)
- Try "Add to Home Screen" flow

**Offline Mode:**
- Install the app
- Turn off WiFi/mobile data
- Open app → should show `/offline` page
- Turn connection back on → should detect and offer reload

**Service Worker:**
- DevTools → Application → Service Workers
- Should show "activated and running"
- Check cached resources under Cache Storage

**Icons:**
- After install, check home screen icon
- Should show orange ₿ symbol on black background

## 🔧 Technical Details

### **PWA Configuration**

**Manifest:** `/public/manifest.json`
- App name: "Cypherpunk BTC"
- Theme color: `#f97316` (Bitcoin orange)
- Background: `#0a0a0a` (Black)
- Display: Standalone (no browser UI)
- Orientation: Portrait (primary)

**Service Worker:** Auto-generated by next-pwa
- Location: `/public/sw.js` (generated on build)
- Strategy: Network-first with offline fallback
- Offline page: `/offline`

**Icons:** 8 sizes (72x72 to 512x512)
- Format: PNG with transparent background
- Design: Orange ₿ on black circle
- Purpose: `any maskable` (adaptive icons)

**Build Mode:** Webpack (required for next-pwa)
```bash
pnpm build  # Uses --webpack flag
```

## 🐛 Troubleshooting

### **"Install" option not showing**

✓ Check HTTPS (PWA requires secure context)
✓ Verify manifest.json is accessible
✓ Check browser console for errors
✓ Try incognito/private mode

### **Offline page not working**

✓ Build production version (dev mode has PWA disabled)
✓ Check service worker is active (DevTools → Application)
✓ Clear browser cache and reinstall

### **Icons not showing**

✓ Check `/public/icon-*.png` files exist
✓ Verify manifest.json has correct paths
✓ Clear home screen cache (reinstall app)

### **Updates not appearing**

✓ Service worker uses `skipWaiting: true` (auto-update)
✓ Close all app instances and reopen
✓ Uninstall and reinstall if needed
