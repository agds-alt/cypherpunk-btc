# 🔐 Cypherpunk BTC

> **Learn Bitcoin from Zero - Following Satoshi's Original Vision**

A comprehensive learning platform to understand Bitcoin from its **philosophical roots**, not as an investment vehicle.

## 🎯 Philosophy

Bitcoin wasn't created to make you rich. It was created to:
- ✊ **Resist centralized control** (central banks, governments)
- 🔐 **Enable financial sovereignty** (be your own bank)
- 📜 **Establish sound money** (fixed supply, no inflation)
- 🌍 **Provide censorship-resistant transactions** (peer-to-peer)

**Price appreciation = bonus, not goal.**
